# Mobile Code Editor

A lightweight, mobile-first code editor built with **HTML**, **CSS**, and **JavaScript (ES Modules)**. This project is designed to run smoothly on Android devices and is being developed using the **SPCK Code Editor**.

## Goals

- Mobile-first design
- Clean and modular code
- Offline support
- Fast performance
- Easy to maintain
- Expandable to support multiple programming languages
- Future desktop support

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)
- Local Storage

## Current Features

- Mobile-first layout
- Dark theme
- Text editor
- New file
- Save file
- Local Storage
- ES Modules (`import` / `export`)
- CSS organized with `@import`
